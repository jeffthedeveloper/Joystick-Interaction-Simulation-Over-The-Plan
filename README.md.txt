# Simulação de Interação no Plano Cartesiano

Este projeto contém gráficos interativos que simulam interações no plano cartesiano.

## 📌 Descrição

Os gráficos demonstram diferentes simulações de interação, incluindo movimentações e posicionamento em quadrantes específicos do plano cartesiano.

## 🚀 Gráficos Implementados

1. **Gráfico 01**: Simulação inicial de interação no plano cartesiano.
   - Arquivo: `grafico01.html`

2. **Gráfico 02**: Simulação de interação focada no **primeiro quadrante**.
   - Arquivo: `grafico2.html`

3. **Gráfico 03**: Simulação de movimentação do eixo do joystick.
   - Posição do Joystick: (-0.4, -0.6) (**terceiro quadrante**)
   - Arquivo: `grafico3.html`
